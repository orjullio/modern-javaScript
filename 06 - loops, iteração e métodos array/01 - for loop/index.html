foor loop

for([expressaoInicial]; [expressaoCondicional]; [expressaoIncrementação])

expressaoInicial: inicializa a variável/conntador

expressaoCondicional: condição onde o loop irá continuar a executar ennquanto a condição não se torna falsa.

expressaoIncrementação: executa após a cada iteração do loop.

O loop for é utilizado para executar um bloco de código um número específico de vezes.

statment: condigo que será executado cada vez que o loop estiver executando.
