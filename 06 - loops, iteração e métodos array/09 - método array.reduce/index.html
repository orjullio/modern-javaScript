Método array.reduce()

O método array.reduce() é utilizado para aplicar uma função a cada elemento de um array, reduzindo-o a um único valor. Ele recebe dois parâmetros: uma função de callback e um valor inicial opcional.

A função de callback recebe quatro parâmetros: o acumulador, o valor atual, o índice do valor atual e o array original.

O método array.reduce() é útil para somar valores, calcular médias, concatenar strings, entre outras operações.

Exemplo de uso

const numeros = [1, 2, 3, 4, 5];
const soma = numeros.reduce((acumulador, valorAtual) => {
    return acumulador + valorAtual;
}, 0);      
console.log(soma); // Saída: 15

No exemplo acima, o método reduce() é utilizado para somar todos os números do array numeros. O valor inicial do acumulador é 0.

Exemplo com objeto

const produtos = [
    { nome: 'Produto 1', preco: 10 },
    { nome: 'Produto 2', preco: 20 },
    { nome: 'Produto 3', preco: 30 }
];
const total = produtos.reduce((acumulador, produto) => {
    return acumulador + produto.preco;
}, 0);
console.log(total); // Saída: 60

No exemplo acima, o método reduce() é utilizado para calcular o preço total de todos os produtos no array produtos. O valor inicial do acumulador é 0.

Exemplo com string

const palavras = ['Olá', ' ', 'mundo', '!'];
const frase = palavras.reduce((acumulador, valorAtual) => {
    return acumulador + valorAtual;
}, '');
console.log(frase); // Saída: 'Olá mundo!'

No exemplo acima, o método reduce() é utilizado para concatenar todas as palavras do array palavras. O valor inicial do acumulador é uma string vazia.

Considerações finais

O método array.reduce() é uma ferramenta poderosa para manipulação de arrays em JavaScript. Ele permite realizar operações complexas de forma concisa e legível.

É importante lembrar que o método reduce() não altera o array original, mas sim retorna um novo valor.

Além disso, o valor inicial do acumulador é opcional. Se não for fornecido, o primeiro elemento do array será utilizado como valor inicial e a iteração começará a partir do segundo elemento.

O método reduce() pode ser utilizado em arrays de qualquer tipo, incluindo números, strings, objetos e até mesmo outros arrays.
